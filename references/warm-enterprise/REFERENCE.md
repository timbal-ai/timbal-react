# warm-enterprise

> Drop `reference.png` beside this file. Until then this trait spec is the

**Design language:** modern warm enterprise (Rippling, Gusto, Mercury). Warm
gray (greige) canvas, ink-navy text, one dignified muted accent, status-heavy
UI (badges, steppers, approvals), soft hairline cards.

**App to generate:** "Ledgerline" — an expense-approval tool: pending approvals
queue, expense detail with receipt panel and approval stepper, and a policy
settings page.

## Trait targets

| Trait | Target |
|---|---|
| Nav paradigm | Standard sidebar with sections; user/org switcher at bottom |
| Surface treatment | Greige canvas (warm ~oklch(0.97, low-chroma warm hue)); white-ish cards, hairline borders |
| Type personality | Humanist sans, calm scale |
| Density | Default-to-comfortable |
| Accent strategy | Muted navy/wine accent; heavy but disciplined use of semantic status tones |
